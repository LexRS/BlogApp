## SwiftUI Blog Client

A sleek, modern blog reader app built with SwiftUI, featuring clean MVVM architecture and seamless API integration with the blog backend service.

## 📱 Overview

A beautifully designed iOS client for consuming blog content, built entirely with SwiftUI. Features real-time updates, smooth navigation, and a native reading experience with dark mode support.

## ✨ Features

Blog Feed - Scroll through latest posts with rich previews
Post Details - Full article view with markdown rendering
User Authentication - Login/Register with secure token storage
Comment System - Read and post comments on articles
Profile Management - View and edit user profile
Bookmarking - Save favorite posts for later reading
Search - Find posts by title, content, or tags
Dark Mode - Seamless light/dark theme switching
Offline Support - Cached content for offline reading

## 🎯 Best Practices Implemented

Single Responsibility - Each component has one clear purpose  
Dependency Injection - Loose coupling and testability  
Reactive State - Combine for reactive UI updates  
Error Handling - Graceful fallbacks and user feedback  
Caching Strategy - Smart cache invalidation  

## 🤝 Integration

This client is designed to work with the [Blog Backend Service](https://github.com/LexRS/blog-api). The API is fully documented with Swagger.
